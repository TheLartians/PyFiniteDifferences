# PyPropagate

A paraxial wave propagation framework for python 2.7 . 

## Installation
    
    git clone --recursive https://github.com/TheLartians/PyPropagate.git
    pip2 install .

## Dependencies

- [Expresso/PyCAS](https://github.com/TheLartians/Expresso)
- boost python
- numpy
- matplotlib
